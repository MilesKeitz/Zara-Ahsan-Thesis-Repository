The following are descriptions for each of the folders/files

Speeches: Contains each of the candidates with the respective speeches transcribed

NLP_Training_Set.xlsx: Training set for NLP that fine-tunes the model to count frequencies of characteristics

Predicted_Frequencies.xlsx: From the NLP, it is the predicted frequencies for each speech for each candidate along with the difference in means for each state

Standardized_Results_T_Tests.csv: Contains the standardized results from each candidate for each speech and is used to conduct t-tests

T-Test_Results.qmd: R script that conducts the t-tests and prints the respective results for each characteristic

nlp_multi.ipynb: Python script of the construction of the NLP and the predicted frequencies for each candidate
